# BOSS-BBS
BOSS BBS is a bulletin board system programmed in the 1980s by Mark Longo using GW‑BASIC.

## BOSS-BBS Features
* User accounts — registration, login, passwords, security levels
* 15 message areas — post, read, reply, delete; public or private
* Full message editor — insert, delete, replace, format lines
* 7 file areas — listings with dates/descriptions, new-file highlights
* File transfers — uploads/downloads via Zmodem (DSZ)
* Door games — launches external programs via DORINFO1.DEF
* SysOp chat — break-in chat with live callers
* Time limits — per-session countdown with warnings
* ANSI graphics — optional color terminal support
* Modem management — auto-init, baud detection, carrier monitoring
